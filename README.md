# 邮件发送助手插件

智能邮件发送插件，支持 LLM 意图判断、多轮对话确认收件人、根据人设生成邮件内容。

## 功能特性

- **智能意图判断**：通过 LLM 自动识别用户是否想发邮件
- **自动识别收件人**：支持 @ 用户自动获取 QQ 号，或消息中提到的 QQ 号/邮箱
- **多轮对话**：逐步确认收件人、邮箱类型、主题、内容
- **灵活收件人**：支持 QQ 号（自动拼接 QQ 邮箱）或直接指定邮箱
- **精致邮件格式**：HTML 邮件模板，渐变色头部，圆角卡片设计
- **内容生成**：调用 LLM 根据主题生成简洁得体的邮件内容
- **命令触发**：支持 `/发邮件` 命令直接触发

## 使用方式

### 方式一：自然语言触发

直接发送包含发邮件意图的消息，例如：
- "给小明发封邮件"
- "帮我发邮件到 test@example.com"
- "给 123456 发邮件"

### 方式二：命令触发

使用 `/发邮件` 命令：
```
/发邮件 123456
/发邮件 test@example.com
/发邮件 123456 明天聚会取消了
```

## 配置说明

### SMTP 配置（必填）

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `smtp.server` | SMTP 服务器地址 | `smtp.qq.com` |
| `smtp.port` | SMTP 端口 | `465` |
| `smtp.email` | 发件邮箱地址 | `your_email@qq.com` |
| `smtp.password` | SMTP 授权码 | `your_authorization_code` |
| `smtp.use_ssl` | 是否使用 SSL | `true` |

### QQ 邮箱 SMTP 配置步骤

1. 登录 QQ 邮箱
2. 进入「设置」→「账户」
3. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务」
4. 开启「POP3/SMTP 服务」或「IMAP/SMTP 服务」
5. 按照提示获取授权码
6. 将授权码填入 `smtp.password` 配置

### LLM 提示词配置（可选）

| 配置项 | 说明 |
|--------|------|
| `llm.intent_prompt` | 意图判断提示词（支持 `{message}` 占位符） |
| `llm.content_prompt` | 邮件内容生成提示词（支持 `{subject}`, `{content_hint}`, `{recipient}` 占位符） |

## 安装方式

1. 将 `email_sender` 文件夹放入 AstrBot 的插件目录
2. 在 AstrBot 管理界面启用插件
3. 配置 SMTP 信息
4. 重启 AstrBot 或热加载插件

## 注意事项

- 首次使用需配置 SMTP 信息，否则无法发送邮件
- QQ 邮箱需要开启 SMTP 服务并获取授权码（不是 QQ 密码）
- 邮件内容由 LLM 生成，可能需要根据实际情况调整提示词
- 插件使用 AstrBot 系统配置的 LLM，请确保已正确配置 AI 服务
